## 배운것

### 01.insertImage.jsp
- <img> 태그 배움

### 02.hyperlink_text.jsp
- <a href>를 배움

### 03.hyperlink_image.jsp
- <a href> 먼저 하고, 태그 안에 <img> 태그를 넣는다. 

### 04.table_default.jsp
- <tr> table-row : 행을 만든다.
- <th> table-header : 컬럼명을 만든다.
- <td> table-data : 데이터 값을 넣는다.

### 05.table_option.jsp
- option 값을 넣는다.
- 테이블 옵션을 넣으면 테이블 모양이 바뀐다.
